# Design Inspiration

Reference screenshots collected to guide the visual direction of the News dashboard UI.

---

## 1. Proposal Progress Card
A clean, minimal stat card with a progress bar pattern — good reference for ticker summary cards and KPI widgets.

![Proposal Progress Card](https://github.com/user-attachments/assets/29dcfd47-ec11-4d40-af5e-f264b42ed158)

**Key takeaways:**
- Light background with soft shadows
- Compact three-column stat layout (sent / interviews / hires)
- Thin horizontal bar charts as micro-visualizations below each stat
- Date selector in top-right corner

---

## 2. Dark Analytics Dashboard (Fitonist)
A full dark-mode SaaS dashboard — closest reference for the News app's multi-panel layout.

![Dark Analytics Dashboard](https://github.com/user-attachments/assets/e3bef07b-287c-4a6c-bc4a-83386557e382)

**Key takeaways:**
- Dark `#1a1a2e`-style background with card grid
- Revenue line chart with two overlapping series (purple + yellow)
- Calendar heatmap for engagement density
- Vertical bar chart for installs by day
- Donut / bubble charts for demographics
- World map for geographic impressions
- Pill-style tab switcher (Today / Week / Month / Range)

---

## 3. Mobile Finance App (iOS)
Minimal iOS widget-style screens — useful for mobile-first card and widget design thinking.

![Mobile Finance App](https://github.com/user-attachments/assets/e40e365e-084f-42df-a5c9-425712ee288e)

**Key takeaways:**
- Muted pastel background colors (sage green, gray, dusty rose)
- Very large typography for primary numbers
- Minimal chrome — almost no borders or lines
- Hatched/textured placeholder areas for charts
- Split-block layout for percentage breakdowns (70% / 30%)

---

## 4. Crextio HR Dashboard
A light-mode enterprise dashboard — reference for onboarding flows, task lists, and calendar integration.

![Crextio HR Dashboard](https://github.com/user-attachments/assets/c3f01ad5-9e54-4d2d-bd9d-ea71d7896764)

**Key takeaways:**
- Warm off-white / cream background
- Top-level KPI strip (Employees / Hires / Projects)
- Work-time progress widget with vertical bar chart
- Circular countdown timer widget
- Onboarding task checklist with completion fraction (2/8)
- Weekly calendar with event blocks
- Dark sidebar panel contrasting with the light main area

## How to use this folder

Add new screenshots here when you find UI patterns worth referencing. Name files descriptively, e.g. `dark_chart_card.png`, and add a short entry to this README with key design takeaways.
