How to write a plugin

A plugin is a single .py file you upload here. It must define these module-level constants and a run() function: 

PLUGIN_NAME        = "My Plugin"       # required
PLUGIN_VERSION     = "1.0.0"           # optional
PLUGIN_TYPE        = "notifier"        # required: notifier | analyzer | screener | exporter
PLUGIN_DESCRIPTION = "What it does"   # optional
PLUGIN_AUTHOR      = "You"            # optional

PLUGIN_SETTINGS = {
    "webhook_url": {"type": "string",   "label": "Webhook URL", "required": True},
    "threshold":   {"type": "number",   "label": "Min confidence", "default": 70},
    "api_key":     {"type": "password", "label": "API Key"},
    "enabled":     {"type": "boolean",  "label": "Active", "default": True},
}

def run(context: dict, settings: dict) -> dict:
    # context for 'notifier': ticker, signal, recommendation, confidence, risk_score
    # context for 'analyzer': ticker, analysis_result
    # context for 'screener': ticker, quant_data
    # context for 'exporter': watchlist, portfolio
    ...
    return {"ok": True, "message": "Done"}

notifier plugins fire automatically when a STRONG_BUY or STRONG_SELL signal is detected.
analyzer plugins run after each Stage 3 analysis.
screener and exporter plugins are triggered manually via the Test button.
